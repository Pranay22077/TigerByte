# TigerByte Concepts

## Core Idea
A toy programming language that supports multilingual commands (English, Mandarin, German).

## Example Commands
- `feed` / `喂` / `füttern`
- `chase` / `追` / `jagen`
- `say` / `说` / `sagen`
